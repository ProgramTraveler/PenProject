### 2021-01-04

#### 绘画时间信息记录

* 信息记录存在问题，笔的间隔太短时会出现0秒的情况，而且时间上的算数运算也存在问题
* 将结果保留两位小数就可以得到正确的结果 **原因** **->** **在计算`/`运算的时候，结果自动保留为整数，而且是向下取整，这就导致了精度的丢失，而产生较大的误差**
